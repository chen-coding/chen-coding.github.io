  • tensorflow: 1.13

安装步骤参照Tensorflow官网安装文档使用 pip 安装 TensorFlow

第一步：检查是否安装Python和pip

如已安装则跳过此步骤，否则安装python和pip（注意版本）

第二步：创建虚拟环境

安装完成后进行测试出现的问题

问题一：dtypes.py

  • 系统版本 Ubuntu20.04 64位

问题一

使用Tensorflow官网安装文档使用 pip 安装 TensorFlow

第一步检查是否安装Python和pip

系统显示都已安装

按照文档提示

跳过这里安装python和pip等

第二步创建虚拟环境

这里开始出错

这里提示要安装python3.8-venv

然而又出错了

百度各种解决办法，都无法解决

然后执行官方文档提示跳过的步骤后问题解决了

成功创建虚拟环境

问题二

由于课程需要安装tensorflow1.13版本，按照tensorflow官方文档操作只能安装2.x的新版本，不支持旧版本的安装

尝试使用命令’’’pip install –upgrade tensorflow==1.13’’’指定安装版本

提示错误找不到1.13版本（提示可用的版本只有2.x）

解决办法：离线安装tensorflow

参考博客(14条消息) 使用 pip 在macos上安装 TensorFlow1.x_joey_ro的博客-CSDN博客

注意这里离线tensorflow文件下载地址要改成对应版本tensorflow · PyPI（这里找1.13版本下载地址，原以为将网址中的1.15.0改成1.13.0就可以了，结果不行，折腾半天发现是1.13.1；另pypi的搜索实在难用，搜索pip install tensorflow==1.13.0[错误]竟然找不到pip install tensorflow==1.13.1[正确]）

下载好上面的文件后准备离线安装，又出错

错误是版本不匹配

原来本机环境是python3.8 这里tensorflow匹配的是python3.7

安装yum

卡在安装yum

ubuntu上装软件还是用apt，yum是别的发行版上的工具
